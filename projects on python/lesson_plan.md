## Урок: "Создаём свой калькулятор"

### Цели
- познакомить детей с идеей программирования и вычислений;
- показать разницу между интерфейсом (HTML) и логикой на Python;
- научить запускать простые программы из терминала;
- шаг за шагом собрать рабочий калькулятор и научить пользоваться им.

### Материалы
- файл `projects on python/calculator.html` — страница с инструкциями (используем как методичку и наглядное пособие);
- файл `projects on python/calculator.py` — готовая программа, которую будем изучать и модифицировать;
- компьютер с установленным Python 3.6+ и доступом к терминалу;
- проект из папки `.cursor/portfolios_for_students`.

---

## Структура занятия (пример на три мини-урока)

### Мини-урок 1. Знакомство с калькулятором (20–25 мин)
1. **Разговор о калькуляторах**  
   - Что делает калькулятор, какие операции знает.  
   - Где дети сталкивались с вычислениями.
2. **Открываем `calculator.html`**  
   - Просмотр разметки, обсуждение шагов запуска.  
   - Показ кнопок «Скопировать команду» — объяснить, зачем нужны терминал и команды.
3. **Подготовка среды**  
   - Запускаем терминал.  
   - Отрабатываем команду `cd "projects on python"`.  
   - Объясняем структуру проекта (есть HTML и Python файлы).
4. **Домашнее задание/мини-задача**  
   - Попросить детей дома повторить шаги и сделать скрин с запущенным калькулятором (без деталей кода).

### Мини-урок 2. Понимаем код Python (25–30 мин)
1. **Повтор запуска**  
   - Все снова открывают HTML-инструкцию и запускают `python calculator.py`.
2. **Обсуждаем поведение программы**  
   - Вводим разные числа, наблюдаем результат.  
   - Пробуем деление на ноль, ввод букв — обсуждаем ошибки.
3. **Разбор кода по частям**  
   - Функция `to_number`: показываем, как текст превращается в число.  
   - Функция `calculate`: изучаем ветвления `if/elif/else`.  
   - `main()`: шаги, вывод текста, чтение ввода.
4. **Мини-задание в классе**  
   - Изменить приветственный текст или добавить ещё одну подсказку.  
   - Запустить и убедиться, что программа работает.

### Мини-урок 3. Добавляем свои идеи (30–35 мин)
1. **Повтор → запуск калькулятора**.  
2. **Расширяем функциональность** (варианты на выбор):  
   - Добавить новую операцию (например, возведение в степень).  
   - Добавить проверку, запрещающую отрицательные числа.  
   - Сделать несколько вычислений подряд (по аналогии с расширенной версией).  
3. **Работа в мини-группах**  
   - Каждая группа придумывает свою «фишку» и показывает результат.  
4. **Финальное обсуждение**  
   - Что понравилось, что было сложно.  
   - Где ещё можно применить Python-калькулятор.

## Советы по дозированной выдаче материала
- **Используйте HTML-страницу как «карту уровня».** Перед каждым этапом просите детей найти нужный шаг на странице и следовать ему.
- **Давайте задания по одному.** Сначала запустить программу, потом изменить текст, потом добавлять новую операцию.
- **Чередуйте демонстрацию и практику.** Сначала показывайте на системе учителя, потом давайте детям повторить.
- **Фиксируйте прогресс.** После каждого шага дети отмечают в тетради или в чек-листе: «запуск — готово», «понял функцию — готово».
- **Поддерживайте инициативу.** Если кто-то справился раньше, предложите придумать новое сообщение, добавить эмодзи или цвета в выводе.
